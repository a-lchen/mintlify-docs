## Bienvenue

Il existe deux façons de créer une documentation API : [OpenAPI](https://mintlify.com/docs/api-playground/openapi/setup) et [composants MDX](https://mintlify.com/docs/api-playground/mdx/configuration). Pour le kit de démarrage, nous utilisons la spécification OpenAPI suivante.

## Authentification

Tous les points de terminaison API sont authentifiés à l'aide de jetons Bearer et sont extraits du fichier de spécification.

```json
"security": [
  {
    "bearerAuth": []
  }
]
```
