## Création d'un snippet personnalisé

**Pré-condition**: Vous devez créer votre fichier snippet dans le `snippets` répertoire.

<Note>
  Toute page dans le `snippets` répertoire sera traitée comme un snippet et ne sera pas
  rendue comme une page autonome. Si vous souhaitez créer une page autonome
  à partir du snippet, importez le snippet dans un autre fichier et appelez-le comme un
  composant.
</Note>

### Export par défaut

1. Ajoutez le contenu que vous souhaitez réutiliser dans votre fichier snippet.
   Vous pouvez éventuellement ajouter des variables qui peuvent être remplies via des props
   lorsque vous importez le snippet.

```mdx snippets/my-snippet.mdx
Hello world! This is my content I want to reuse across pages. My keyword of the
day is {word}.
```

<Warning>
  Le contenu que vous souhaitez réutiliser doit être dans le `snippets` répertoire pour
  que l'import fonctionne.
</Warning>

2. Importez le snippet dans votre fichier de destination.

```mdx destination-file.mdx
---
title: My title
description: My Description
---

import MySnippet from '/snippets/path/to/my-snippet.mdx';

## Header

Lorem impsum dolor sit amet.

<MySnippet word="bananas" />
```

### Variables réutilisables

1. Exportez une variable depuis votre fichier snippet :

```mdx snippets/path/to/custom-variables.mdx
export const myName = 'my name';

export const myObject = { fruit: 'strawberries' };
```

2. Importez le snippet depuis votre fichier de destination et utilisez la variable :

```mdx destination-file.mdx
---
title: My title
description: My Description
---

import { myName, myObject } from '/snippets/path/to/custom-variables.mdx';

Hello, my name is {myName} and I like {myObject.fruit}.
```

### Composants réutilisables

1. Dans votre fichier snippet, créez un composant qui accepte des props en exportant
   votre composant sous forme de fonction fléchée.

```mdx snippets/custom-component.mdx
export const MyComponent = ({ title }) => (
  <div>
    <h1>{title}</h1>
    <p>... snippet content ...</p>
  </div>
);
```

<Warning>
  MDX ne se compile pas dans le corps d'une fonction fléchée. Tenez-vous en à la syntaxe
  HTML quand vous le pouvez ou utilisez un export par défaut si vous avez besoin d'utiliser MDX.
</Warning>

2. Importez le snippet dans votre fichier de destination et passez les props

```mdx destination-file.mdx
---
title: My title
description: My Description
---

import { MyComponent } from '/snippets/custom-component.mdx';

Lorem ipsum dolor sit amet.

<MyComponent title={'Custom title'} />
```
