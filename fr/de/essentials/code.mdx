### Code en Ligne

Pour désigner`word`ou`phrase`comme code, entourez-le de backticks (\`).

```
To denote a `word` or `phrase` as code, enclose it in backticks (`).
```

### Bloc de Code

Utilisez[les blocs de code délimités](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks)en entourant le code de trois backticks et en faisant suivre les backticks initiaux du langage de programmation de votre extrait pour obtenir la coloration syntaxique. En option, vous pouvez également écrire le nom de votre code après le langage de programmation.

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
