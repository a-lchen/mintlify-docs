## Configuration de l'environnement de développement

Découvrez comment mettre à jour votre documentation localement et la déployer publiquement.

### Édition et aperçu

<AccordionGroup>
  <Accordion icon="github" title="Clone your docs locally">
    Pendant le processus d'intégration, nous avons créé un dépôt sur votre Github avec
    vos contenus de documentation. Vous pouvez trouver ce dépôt dans notre
    [Tableau de bord](https://dashboard.mintlify.com). Pour cloner le dépôt
    localement, suivez ces
    [instructions](https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository)
    dans votre terminal.
  </Accordion>

  <Accordion icon="rectangle-terminal" title="Preview changes">
    L'aperçu vous aide à vous assurer que vos modifications apparaissent comme prévu. Nous avons développé
    une interface en ligne de commande pour visualiser ces changements localement.

    1. Installez le
       [Mintlify CLI](https://www.npmjs.com/package/mintlify) pour visualiser
       les changements de documentation localement avec cette commande :`npm i -g mintlify`
    2. Exécutez la commande suivante dans le répertoire principal de votre documentation (où se trouve`docs.json`) :`mintlify dev`

    <Note>
      Si vous utilisez actuellement le fichier de configuration`mint.json` hérité, veuillez mettre à jour le Mintlify CLI :

      `npm i -g mintlify@latest`
      Et exécutez la nouvelle commande de mise à niveau dans votre dépôt de documentation :

      `mintlify upgrade`
      Vous devriez maintenant utiliser le nouveau`docs.json` fichier de configuration. Vous pouvez supprimer le fichier`mint.json` de votre dépôt.
    </Note>
  </Accordion>
</AccordionGroup>

### Déployer les modifications

<AccordionGroup>
  <Accordion icon="message-bot" title="Install our Github app">
    Notre application Github déploie automatiquement vos modifications sur votre page de documentation,
    vous n'avez donc pas à vous soucier des déploiements. Vous pouvez trouver le lien d'installation dans
    votre[Tableau de bord](https://dashboard.mintlify.com). Une fois le bot installé avec succès,
    une coche devrait apparaître à côté du hash de commit du dépôt.
  </Accordion>

  <Accordion icon="rocket" title="Push your changes">
    [Committez et poussez vos modifications vers
    Git](https://docs.github.com/en/get-started/using-git/pushing-commits-to-a-remote-repository#about-git-push)
    pour que vos modifications soient mises à jour sur votre page de documentation. Si vous poussez et ne voyez pas
    l'application Github déployer vos modifications avec succès, vous pouvez également mettre à jour votre documentation manuellement
    via notre[Tableau de bord](https://dashboard.mintlify.com).
  </Accordion>
</AccordionGroup>
