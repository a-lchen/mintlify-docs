## 图片

### 使用 Markdown

这[markdown 语法](https://www.markdownguide.org/basic-syntax/#images) 让您使用以下代码添加图片

```md
![title](/path/image.jpg)
```

请注意，图像文件的大小必须小于5MB。否则，我们建议托管在类似 Mintlify 的服务上。[Cloudinary](https://cloudinary.com/) 或 [S3](https://aws.amazon.com/s3/)。然后，您可以使用该 URL 并嵌入。

### 使用嵌入

要对图像进行更多自定义，您还可以使用[嵌入](/writing-content/embed)添加图片

```html
<img height="200" src="/path/image.jpg" />
```

### iFrames

在文档中加载另一个HTML页面。最常用于嵌入视频。

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
