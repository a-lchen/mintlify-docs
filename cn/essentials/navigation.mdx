导航菜单是每个网站上链接的列表。

您可能会更新`docs.json`每次添加新页面时。页面不会自动显示。

## 导航语法

我们的导航语法是递归的，这意味着您可以创建嵌套的导航组。您不需要包含`.mdx`在页面名称中。

<CodeGroup>
  ```json Regular Navigation
  "navigation": {
    "tabs": [
      {
        "tab": "Docs",
        "groups": [
          {
            "group": "Getting Started",
            "pages": ["quickstart"]
          }
        ]
      }
    ]
  }
  ```

  ```json Nested Navigation
  "navigation": {
    "tabs": [
      {
        "tab": "Docs",
        "groups": [
          {
            "group": "Getting Started",
            "pages": [
              "quickstart",
              {
                "group": "Nested Reference Pages",
                "pages": ["nested-reference-page"]
              }
            ]
          }
        ]
      }
    ]
  }
  ```
</CodeGroup>

## 文件夹

只需将您的MDX文件放入文件夹并更新路径`docs.json`。

例如，要在该页面`https://yoursite.com/your-folder/your-page`你将创建一个名为`your-folder`包含一个名为 MDX 的文件`your-page.mdx`.

<Warning>
  你不能使用`api`除非您将其嵌套在另一个文件夹中，否则文件夹名称无需如此命名。Mintlify 使用 Next.js，其保留了顶层`api`用于内部服务器调用的文件夹。例如，文件夹名称如`api-reference` 可以接受。
</Warning>

```json Navigation With Folder
"navigation": {
  "tabs": [
    {
      "tab": "Docs",
      "groups": [
        {
          "group": "Group Name",
          "pages": ["your-folder/your-page"]
        }
      ]
    }
  ]
}
```

## 隐藏页面

MDX 文件未包含在 `docs.json`不会显示在侧边栏中，但可以通过搜索栏或直接链接访问它们。
