## 基本基本测试测试

### 内联代码

为了一个`word`或`phrase`要将其标记为代码，请将其用反引号 (\`) 括起来。

```
To denote a `word` or `phrase` as code, enclose it in backticks (`).
```

### 代码块

使用[围栏代码块](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks)通过将代码包含在三个反引号中，并在起始反引号后面添加您的代码片段的编程语言来实现语法高亮显示。您也可以选择在编程语言后面写上您的代码名称。

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
