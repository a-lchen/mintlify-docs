导航菜单是每个网站上的链接列表。

您可能会`docs.json`在每次添加新页面时进行更新。页面不会自动显示。

## 导航语法

我们的导航语法是递归的，这意味着您可以创建嵌套的导航组。您不需要`.mdx`包含在页面名称中。

<CodeGroup>
  ```json Regular Navigation
  "navigation": {
    "tabs": [
      {
        "tab": "Docs",
        "groups": [
          {
            "group": "Getting Started",
            "pages": ["quickstart"]
          }
        ]
      }
    ]
  }
  ```

  ```json Nested Navigation
  "navigation": {
    "tabs": [
      {
        "tab": "Docs",
        "groups": [
          {
            "group": "Getting Started",
            "pages": [
              "quickstart",
              {
                "group": "Nested Reference Pages",
                "pages": ["nested-reference-page"]
              }
            ]
          }
        ]
      }
    ]
  }
  ```
</CodeGroup>

## 文件夹

只需将您的MDX文件放在文件夹中并更新`docs.json`中的路径。

例如，要在`https://yoursite.com/your-folder/your-page`下创建页面，您需要创建一个名为`your-folder`的文件夹，其中包含一个名为`your-page.mdx`的MDX文件。

<Warning>
  您不能`api`将其用作顶层文件夹名称，除非将其嵌套在另一个文件夹中。Mintlify使用Next.js，它保留顶层`api`文件夹用于内部服务器调用。像`api-reference`这样的文件夹名称是可以接受的。
</Warning>

```json Navigation With Folder
"navigation": {
  "tabs": [
    {
      "tab": "Docs",
      "groups": [
        {
          "group": "Group Name",
          "pages": ["your-folder/your-page"]
        }
      ]
    }
  ]
}
```

## 隐藏页面

未包含在`docs.json`中的MDX文件不会显示在侧边栏中，但可以通过搜索栏和直接链接访问。
