导航菜单是每个网站上链接的列表。

您可能会`docs.json`每次添加新页面时更新。 页面不会自动出现。

## 导航语法

我们的导航语法是递归的，这意味着您可以创建嵌套的导航组。您不需要`.mdx`在页面名称中包含。

<CodeGroup>
  ```json Regular Navigation
  "navigation": {
    "tabs": [
      {
        "tab": "Docs",
        "groups": [
          {
            "group": "Getting Started",
            "pages": ["quickstart"]
          }
        ]
      }
    ]
  }
  ```

  ```json Nested Navigation
  "navigation": {
    "tabs": [
      {
        "tab": "Docs",
        "groups": [
          {
            "group": "Getting Started",
            "pages": [
              "quickstart",
              {
                "group": "Nested Reference Pages",
                "pages": ["nested-reference-page"]
              }
            ]
          }
        ]
      }
    ]
  }
  ```
</CodeGroup>

## 文件夹

只需将您的 MDX 文件放在文件夹中并更新路径到`docs.json`.

例如，若要将页面放在`https://yoursite.com/your-folder/your-page`您可以拥有一个名为`your-folder`创建一个名为的MDX文件`your-page.mdx`enthält.

<Warning>
  你不能`api`作为文件夹的名称使用，除非您将其嵌套在另一个文件夹中。Mintlify 使用 Next.js，它的顶级`api`保留用于内部服务器调用的文件夹。 一个文件夹名称如`api-reference`将被接受。
</Warning>

```json Navigation With Folder
"navigation": {
  "tabs": [
    {
      "tab": "Docs",
      "groups": [
        {
          "group": "Group Name",
          "pages": ["your-folder/your-page"]
        }
      ]
    }
  ]
}
```

## 隐藏页面

未在`docs.json`包含在其中的内容不会显示在侧栏中，但可以通过搜索栏和直接链接访问。
