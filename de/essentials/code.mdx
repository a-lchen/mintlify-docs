### Inline-Code

Um einen`word`oder`phrase`als Code zu kennzeichnen, schließen Sie ihn in Backticks (\`) ein.

```
To denote a `word` or `phrase` as code, enclose it in backticks (`).
```

### Codeblock

Verwenden Sie[umzäunte Codeblöcke](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks)indem Sie den Code in drei Backticks einschließen und den führenden Backticks die Programmiersprache Ihres Snippets folgen lassen, um Syntaxhervorhebung zu erhalten. Optional können Sie auch den Namen Ihres Codes nach der Programmiersprache schreiben.

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
