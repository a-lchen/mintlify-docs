### Código Inline

Para denotar um `word` ou `phrase` como código, coloque-o entre crases (\`).

```
To denote a `word` or `phrase` as code, enclose it in backticks (`).
```

### Bloco de Código

Use [blocos de código cercados](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks) colocando o código entre três crases e seguindo as crases iniciais com a linguagem de programação do seu trecho para obter destaque de sintaxe. Opcionalmente, você também pode escrever o nome do seu código após a linguagem de programação.

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
