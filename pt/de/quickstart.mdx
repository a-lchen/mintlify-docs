## Configurar ambiente de desenvolvimento

Aprenda como atualizar sua documentação localmente e implantá-la publicamente.

### Editar e Visualizar

<AccordionGroup>
  <Accordion icon="github" title="Clone your docs locally">
    Durante o processo de integração, criamos um repositório no seu Github com
    o conteúdo da sua documentação. Você pode encontrar este repositório no nosso
    [Painel](https://dashboard.mintlify.com). Para clonar o repositório
    localmente, siga estas[instruções](https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository)
    no seu terminal.
  </Accordion>

  <Accordion icon="rectangle-terminal" title="Preview changes">
    A visualização ajuda a garantir que suas alterações apareçam como pretendido. Desenvolvemos uma
    interface de linha de comando para visualizar essas alterações localmente.

    1. Instale o[Mintlify CLI](https://www.npmjs.com/package/mintlify) para visualizar as
       alterações da documentação localmente com este comando:`npm i -g mintlify`
    2. Execute o seguinte comando no diretório raiz da sua documentação (onde`docs.json` é): `mintlify dev`

    <Note>
      Se você estiver usando atualmente o `mint.json` arquivo de configuração legado, por favor atualize o Mintlify CLI:

      `npm i -g mintlify@latest`
      E execute o novo comando de atualização em seu repositório de documentação:

      `mintlify upgrade`
      Agora você deve estar usando o novo `docs.json` arquivo de configuração. Você pode excluir o `mint.json` arquivo do seu repositório.
    </Note>
  </Accordion>
</AccordionGroup>

### Implantar Alterações

<AccordionGroup>
  <Accordion icon="message-bot" title="Install our Github app">
    Nosso aplicativo Github implanta automaticamente suas alterações em sua página de documentação, então você não precisa se preocupar com implantações. Você pode encontrar o link de instalação no seu [Dashboard](https://dashboard.mintlify.com). Uma vez que o bot for instalado com sucesso, você deve ver uma marca de seleção ao lado do hash do commit do repositório.
  </Accordion>

  <Accordion icon="rocket" title="Push your changes">
    [Faça commit e push das suas alterações para o Git](https://docs.github.com/en/get-started/using-git/pushing-commits-to-a-remote-repository#about-git-push)
    para que suas alterações sejam atualizadas em sua página de documentação. Se você fizer push e não ver o aplicativo Github implantando suas alterações com sucesso, você também pode atualizar sua documentação manualmente através do nosso [Dashboard](https://dashboard.mintlify.com).
  </Accordion>
</AccordionGroup>
