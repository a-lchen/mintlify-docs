### Código Inline

Para marcar`word`ou`phrase`como código, envolva-o em acentos graves (\`).

```
To denote a `word` or `phrase` as code, enclose it in backticks (`).
```

### Bloco de Código

Use[blocos de código cercados](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks)envolvendo o código em três acentos graves e seguindo os acentos graves iniciais com a linguagem de programação do seu snippet para realce de sintaxe. Opcionalmente, você também pode escrever o nome do seu código após a linguagem de programação.

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
