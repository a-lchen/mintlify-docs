Siga estas etapas para instalar e executar o Mintlify no seu sistema operacional:

**Passo 1**: Instale o Mintlify:

<CodeGroup>
  ```bash npm
  npm i -g mintlify
  ```

  ```bash yarn
  yarn global add mintlify
  ```
</CodeGroup>

**Passo 2**: Navegue até o diretório da documentação (onde está o arquivo`docs.json`) e execute o seguinte comando:

```bash
mintlify dev
```

Uma prévia local da sua documentação estará disponível em`http://localhost:3000`.

### Portas Personalizadas

Por padrão, o Mintlify usa a porta 3000. Você pode personalizar a porta em que o Mintlify roda usando a flag`--port`. Por exemplo, para rodar o Mintlify na porta 3333, use este comando:

```bash
mintlify dev --port 3333
```

Se você tentar rodar o Mintlify em uma porta que já está em uso, a próxima porta disponível será usada:

```md
Port 3000 is already in use. Trying 3001 instead.
```

## Versões do Mintlify

Por favor, note que cada versão do CLI está vinculada a uma versão específica do Mintlify. Se seu site local não corresponder à versão de produção, por favor atualize o CLI:

<CodeGroup>
  ```bash npm
  npm i -g mintlify@latest
  ```

  ```bash yarn
  yarn global upgrade mintlify
  ```
</CodeGroup>

## Verificando Links

O CLI pode ajudar a verificar links de referência em sua documentação. Para identificar links quebrados, use o seguinte comando:

```bash
mintlify broken-links
```

## Formatação de Código

Recomendamos usar extensões em sua IDE para detectar e formatar MDX. Se você é usuário do VSCode, considere a [extensão MDX VSCode](https://marketplace.visualstudio.com/items?itemName=unifiedjs.vscode-mdx) para realce de sintaxe e [Prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode) para formatação de código.

## Solução de Problemas

<AccordionGroup>
  <Accordion title="Error: Could not load the &#x22;sharp&#x22; module using the darwin-arm64 runtime">
    Isso pode ser devido a uma versão desatualizada do Node. Tente o seguinte:

    1. Remova a versão atualmente instalada do Mintlify: `npm remove -g mintlify`
    2. Atualize para o Node v19 ou superior.
    3. Reinstale o Mintlify: `npm install -g mintlify`
  </Accordion>

  <Accordion title="Issue: Encountering an unknown error">
    Solução: Vá para o diretório raiz do seu dispositivo e exclua a pasta \~/.mintlify. Depois disso, execute `mintlify dev` novamente.
  </Accordion>
</AccordionGroup>

Curioso sobre o que mudou na versão do CLI? [Confira o registro de alterações do CLI.](https://www.npmjs.com/package/mintlify?activeTab=versions)
