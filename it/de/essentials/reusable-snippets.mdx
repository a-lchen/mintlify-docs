## Creazione di uno snippet personalizzato

**Prerequisito**: Devi creare il tuo file snippet nella`snippets`directory.

<Note>
  Ogni pagina nella`snippets`directory viene trattata come uno snippet e non viene renderizzata come pagina autonoma. Se desideri creare una pagina autonoma dallo snippet, importa lo snippet in un altro file e chiamalo come componente.
</Note>

### Export predefinito

1. Aggiungi al tuo file snippet i contenuti che desideri riutilizzare in più luoghi. Opzionalmente, puoi aggiungere variabili che possono essere popolate tramite props quando importi lo snippet.

```mdx snippets/my-snippet.mdx
Hello world! This is my content I want to reuse across pages. My keyword of the
day is {word}.
```

<Warning>
  Il contenuto che desideri riutilizzare deve essere nella`snippets`directory affinché l'importazione funzioni.
</Warning>

2. Importa lo snippet nel tuo file di destinazione.

```mdx destination-file.mdx
---
title: My title
description: My Description
---

import MySnippet from '/snippets/path/to/my-snippet.mdx';

## Header

Lorem impsum dolor sit amet.

<MySnippet word="bananas" />
```

### Variabili riutilizzabili

1. Esporta una variabile dal tuo file snippet:

```mdx snippets/path/to/custom-variables.mdx
export const myName = 'my name';

export const myObject = { fruit: 'strawberries' };
```

2. Importa lo snippet dal tuo file di destinazione e usa la variabile:

```mdx destination-file.mdx
---
title: My title
description: My Description
---

import { myName, myObject } from '/snippets/path/to/custom-variables.mdx';

Hello, my name is {myName} and I like {myObject.fruit}.
```

### Componenti Riutilizzabili

1. Nel tuo file snippet, crea un componente che accetta props esportandolo come funzione freccia.

```mdx snippets/custom-component.mdx
export const MyComponent = ({ title }) => (
  <div>
    <h1>{title}</h1>
    <p>... snippet content ...</p>
  </div>
);
```

<Warning>
  MDX non viene compilato nel corpo di una funzione freccia. Attieniti alla sintassi HTML quando possibile o usa un'esportazione predefinita se hai bisogno di MDX.
</Warning>

2. Importa lo snippet nel tuo file di destinazione e passa le props

```mdx destination-file.mdx
---
title: My title
description: My Description
---

import { MyComponent } from '/snippets/custom-component.mdx';

Lorem ipsum dolor sit amet.

<MyComponent title={'Custom title'} />
```
