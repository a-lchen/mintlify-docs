## Immagine

### Utilizzo di Markdown

La [sintassi Markdown](https://www.markdownguide.org/basic-syntax/#images) permette di aggiungere immagini con il seguente codice

```md
![title](/path/image.jpg)
```

Nota che la dimensione del file immagine deve essere inferiore a 5MB. Altrimenti, consigliamo di ospitare su un servizio come [Cloudinary](https://cloudinary.com/) o [S3](https://aws.amazon.com/s3/). Puoi quindi utilizzare quell'URL e incorporarlo.

### Utilizzo degli incorporamenti

Per ottenere maggiore personalizzazione con le immagini, puoi anche utilizzare [incorporamenti](/writing-content/embed) per aggiungere immagini

```html
<img height="200" src="/path/image.jpg" />
```

### iFrames

Carica un'altra pagina HTML all'interno del documento. Utilizzato più comunemente per incorporare video.

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
