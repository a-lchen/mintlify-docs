## Configura il tuo sviluppo

Impara come aggiornare la tua documentazione localmente e pubblicarla pubblicamente.

### Modifica e anteprima

<AccordionGroup>
  <Accordion icon="github" title="Clone your docs locally">
    Durante il processo di onboarding, abbiamo creato un repository su Github con
    il contenuto della tua documentazione. Puoi trovare questo repository sul nostro
    [dashboard](https://dashboard.mintlify.com). Per clonare il repository
    localmente, segui queste
    [istruzioni](https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository)
    nel tuo terminale.
  </Accordion>

  <Accordion icon="rectangle-terminal" title="Preview changes">
    L'anteprima ti aiuta ad assicurarti che le tue modifiche appaiano come previsto. Abbiamo costruito un'interfaccia a riga di comando per visualizzare queste modifiche localmente.

    1. Installa il
       [Mintlify CLI](https://www.npmjs.com/package/mintlify) per visualizzare in anteprima le
       modifiche alla documentazione localmente con questo comando: `npm i -g mintlify`
    2. Esegui il seguente comando nella root della tua documentazione (dove`docs.json` si trova): `mintlify dev`

    <Note>
      Se stai attualmente utilizzando il legacy `mint.json` file di configurazione, per favore aggiorna il Mintlify CLI:

      `npm i -g mintlify@latest`
      Ed esegui il nuovo comando di aggiornamento nel tuo repository della documentazione:

      `mintlify upgrade`
      Ora dovresti utilizzare il nuovo `docs.json` file di configurazione. Sentiti libero di eliminare il `mint.json` file dal tuo repository.
    </Note>
  </Accordion>
</AccordionGroup>

### Distribuisci le tue modifiche

<AccordionGroup>
  <Accordion icon="message-bot" title="Install our Github app">
    La nostra app Github distribuisce automaticamente le tue modifiche al tuo sito di documentazione, quindi non devi gestire le distribuzioni da solo. Puoi trovare il link per l'installazione sul tuo [dashboard](https://dashboard.mintlify.com). Una volta che il bot è stato
    installato con successo, dovrebbe esserci un segno di spunta accanto all'hash del commit
    del repository.
  </Accordion>

  <Accordion icon="rocket" title="Push your changes">
    [Esegui il commit e il push delle tue modifiche su
    Git](https://docs.github.com/en/get-started/using-git/pushing-commits-to-a-remote-repository#about-git-push)
    per aggiornare le tue modifiche nel tuo sito di documentazione. Se fai il push e non vedi che
    l'app Github ha distribuito con successo le tue modifiche, puoi anche aggiornare
    manualmente la tua documentazione attraverso il nostro [dashboard](https://dashboard.mintlify.com).
  </Accordion>
</AccordionGroup>
