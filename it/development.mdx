<Info>
  **Prerequisito**: Si prega di installare Node.js (versione 19 o superiore) prima di procedere.<br />
  Si prega di aggiornare a `docs.json` prima di procedere ed eliminare il file legacy `mint.json`.
</Info>

Segui questi passaggi per installare ed eseguire Mintlify sul tuo sistema operativo:

**Passo 1**: Installa Mintlify:

<CodeGroup>
  ```bash npm
  npm i -g mintlify
  ```

  ```bash yarn
  yarn global add mintlify
  ```
</CodeGroup>

**Passo 2**: Naviga nella directory dei documenti (dove si trova il file `docs.json`) ed esegui il seguente comando:

```bash
mintlify dev
```

Un'anteprima locale della tua documentazione sarà disponibile su `http://localhost:3000`.

### Porte Personalizzate

Per impostazione predefinita, Mintlify usa la porta 3000. Puoi personalizzare la porta su cui Mintlify viene eseguito usando il flag `--port`. Per eseguire Mintlify sulla porta 3333, ad esempio, usa questo comando:

```bash
mintlify dev --port 3333
```

Se tenti di eseguire Mintlify su una porta già in uso, utilizzerà la prossima porta disponibile:

```md
Port 3000 is already in use. Trying 3001 instead.
```

## Versioni di Mintlify

Si prega di notare che ogni release della CLI è associata a una specifica versione di Mintlify. Se il tuo sito web locale non corrisponde alla versione di produzione, aggiorna la CLI:

<CodeGroup>
  ```bash npm
  npm i -g mintlify@latest
  ```

  ```bash yarn
  yarn global upgrade mintlify
  ```
</CodeGroup>

## Validazione dei Link

La CLI può aiutare con la validazione dei link di riferimento nella tua documentazione. Per identificare eventuali link non funzionanti, usa il seguente comando:

```bash
mintlify broken-links
```

Suggeriamo di utilizzare estensioni sul tuo IDE per riconoscere e formattare MDX. Se sei un utente VSCode, considera l'[estensione MDX VSCode](https://marketplace.visualstudio.com/items?itemName=unifiedjs.vscode-mdx) per l'evidenziazione della sintassi, e [Prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode) per la formattazione del codice.

## Risoluzione dei Problemi

<AccordionGroup>
  <Accordion title="Error: Could not load the &#x22;sharp&#x22; module using the darwin-arm64 runtime">
    Questo potrebbe essere dovuto a una versione obsoleta di node. Prova quanto segue:

    1. Rimuovi la versione attualmente installata di mintlify: `npm remove -g mintlify`
    2. Aggiorna a Node v19 o superiore.
    3. Reinstalla mintlify: `npm install -g mintlify`
  </Accordion>

  <Accordion title="Issue: Encountering an unknown error">
    Soluzione: Vai alla root del tuo dispositivo ed elimina la cartella \~/.mintlify. Successivamente, esegui `mintlify dev` di nuovo.
  </Accordion>
</AccordionGroup>

Curioso di sapere cosa è cambiato nella versione CLI? [Controlla il changelog della CLI.](https://www.npmjs.com/package/mintlify?activeTab=versions)
