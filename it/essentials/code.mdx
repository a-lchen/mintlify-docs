### Codice Inline

Per indicare un `word` o `phrase` come codice, racchiudilo tra backtick (\`).

```
To denote a `word` or `phrase` as code, enclose it in backticks (`).
```

### Blocco di Codice

Usa [blocchi di codice delimitati](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks) racchiudendo il codice tra tre backtick e facendo seguire i backtick iniziali con il linguaggio di programmazione del tuo snippet per ottenere l'evidenziazione della sintassi. Opzionalmente, puoi anche scrivere il nome del tuo codice dopo il linguaggio di programmazione.

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
