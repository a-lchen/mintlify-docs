## Creazione di uno snippet personalizzato

**Precondizione**: Devi creare il tuo file snippet nella directory `snippets`.

<Note>
  Qualsiasi pagina nella directory `snippets` verrà trattata come uno snippet e non
  sarà renderizzata come pagina autonoma. Se vuoi creare una pagina autonoma
  dallo snippet, importa lo snippet in un altro file e chiamalo come
  componente.
</Note>

### Export predefinito

1. Aggiungi al tuo file snippet il contenuto che vuoi riutilizzare in più
   posizioni. Opzionalmente, puoi aggiungere variabili che possono essere compilate tramite props
   quando importi lo snippet.

```mdx snippets/my-snippet.mdx
Hello world! This is my content I want to reuse across pages. My keyword of the
day is {word}.
```

<Warning>
  Il contenuto che vuoi riutilizzare deve essere all'interno della directory `snippets` affinché
  l'importazione funzioni.
</Warning>

2. Importa lo snippet nel tuo file di destinazione.

```mdx destination-file.mdx
---
title: My title
description: My Description
---

import MySnippet from '/snippets/path/to/my-snippet.mdx';

## Header

Lorem impsum dolor sit amet.

<MySnippet word="bananas" />
```

### Variabili riutilizzabili

1. Esporta una variabile dal tuo file snippet:

```mdx snippets/path/to/custom-variables.mdx
export const myName = 'my name';

export const myObject = { fruit: 'strawberries' };
```

2. Importa lo snippet dal tuo file di destinazione e usa la variabile:

```mdx destination-file.mdx
---
title: My title
description: My Description
---

import { myName, myObject } from '/snippets/path/to/custom-variables.mdx';

Hello, my name is {myName} and I like {myObject.fruit}.
```

### Componenti riutilizzabili

1. Nel tuo file snippet, crea un componente che accetta props esportando
   il tuo componente sotto forma di arrow function.

```mdx snippets/custom-component.mdx
export const MyComponent = ({ title }) => (
  <div>
    <h1>{title}</h1>
    <p>... snippet content ...</p>
  </div>
);
```

<Warning>
  MDX non compila all'interno del corpo di una arrow function. Attieniti alla sintassi
  HTML quando possibile o usa un export predefinito se hai bisogno di usare MDX.
</Warning>

2. Importa lo snippet nel tuo file di destinazione e passa le props

```mdx destination-file.mdx
---
title: My title
description: My Description
---

import { MyComponent } from '/snippets/custom-component.mdx';

Lorem ipsum dolor sit amet.

<MyComponent title={'Custom title'} />
```
